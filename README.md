# Javascript Mini Projects
Welcome to my repository of JavaScript mini-projects! This is a dedicated space where I document my web development journey, focusing on mastering core JavaScript concepts, DOM manipulation, state management, and modern CSS UI design.
# Counter
**🌟 Key Features**
- State Persistence: Uses localStorage to remember your count even if you refresh or close the browser tab.
- Modern UI/UX: Styled using advanced CSS techniques including glassmorphism, gradients, and tactile button transitions.
- DOM Event Handling: Built using the DOMContentLoaded event listener to ensure safe and optimized script execution.

**🛠️ Concepts Learned**
- Reading and writing data with the Web Storage API (localStorage).
- Grabbing elements and updating UI text dynamically using document.querySelector.
- Creating interactive CSS hover/active states with CSS transitions and transforms.
